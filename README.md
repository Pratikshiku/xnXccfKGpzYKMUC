## SpringBoot Online Mall Shopping System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4fd8accbb8994f6bb9c0edea68423bee/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
Background Section:
1. Login
2. Register
3. Change Password
4. Personal Information
5. Order Reviews
6. My Favorites Management
7. Order Management
Frontend Section:
8. Homepage
9. Product Information
10. Product News
11. Personal Center
12. Background Management
13. Shopping Cart
14. Online Customer Service

**Administrator Functions:**
1. Login
2. Change Password
3. Personal Information
4. User Management
5. Product Category Management
6. Product Information Management
7. Order Review Management
8. System Management (online customer service, carousel, product news)
9. Order Management
